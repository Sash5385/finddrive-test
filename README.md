Ізольований клон для перевірки монетизаційних змін вживу, без ризику для
`finddrive.in.ua`. Джерело: гілка `claude/monetization-phase01` репозиторію
`sash5385/finddrive` (Фаза 0 + 0.1, див. `MONETIZATION.md`).

**Увага:** цей сайт використовує ту саму Firebase-базу (Firestore/Auth), що й
production — це не пісочниця з окремими даними, а лише окрема адреса й окремий
код для клікабельного тестування UI/UX.

Не деплоїться автоматично в основний сайт. Видалити, коли тестування завершено.
